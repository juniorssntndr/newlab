# Estudio de mercado dental - 4 formularios balanceados

Este paquete contiene los insumos para levantar entrevistas presenciales de 5-7 minutos con odontólogos o consultorios dentales en Arequipa.

## Archivos

- `formularios_estudio_mercado_dental.csv`: matriz maestra para importar o copiar a Google Sheets.
- `crear_google_forms_estudio_mercado_dental.gs`: script para crear automáticamente los 4 Google Forms.
- `matriz_maestra_estudio_mercado_dental.xlsx`: versión editable y más cómoda para revisar en Excel o Google Sheets.

## Cómo crear los Google Forms

1. Entra a https://script.google.com/.
2. Crea un proyecto nuevo.
3. Pega el contenido completo de `crear_google_forms_estudio_mercado_dental.gs`.
4. Ejecuta `crearFormulariosEstudioMercadoDental`.
5. Acepta los permisos de Google.
6. Abre los registros de ejecución para copiar las URLs editables, las URLs públicas y el Google Sheets de respuestas.

## Uso recomendado en campo

- Haz una entrevista por doctor usando solo uno de los formularios.
- Alterna los formularios A, B, C y D por zona para no sesgar el estudio.
- No expliques todavía la solución completa. Primero valida comportamiento, dolores y criterios de decisión.
- Mantén la entrevista debajo de 7 minutos. Si el doctor quiere hablar más, registra el comentario en la pregunta abierta.

## Lectura de resultados

Después de recolectar respuestas, analiza por:

- Zona o distrito.
- Nivel de adopción digital.
- Frecuencia de uso de laboratorio.
- Materiales y servicios más solicitados.
- Dolores técnicos o logísticos repetidos.
- Palancas de cambio de proveedor.
- Interés real en seguimiento digital y servicios innovadores.
